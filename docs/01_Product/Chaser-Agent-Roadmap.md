# Chaser agent Roadmap

## Phase 0 — Scaffold
Docs, safe toy datasets, importable package, deterministic source-card stub, and smoke tests.

## Phase 1 — Source card harness
Turn source text into claims, uncertainty labels, action candidates, and memory candidates with no external API calls by default.

## Phase 2 — Eval harness
Add scoring, run logs, regression snapshots, and human review exports.

## Phase 3 — Adapter experiments
Test provider, local model, MCP, Hermes, and OpenClaw adapter boundaries without granting production authority.

## Phase 4 — Reviewed datasets
Curate private and public golden sets separately. Fine-tuning candidates require separate approval.
