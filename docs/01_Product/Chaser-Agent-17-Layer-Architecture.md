# Chaser agent 17-Layer Architecture

Chaser agent inherits the ChaseOS layered worldview but implements only bounded product surfaces until eval evidence and operator approval expand scope.

Purpose: define the product concern for user / operator layer.

ChaseOS source: parent architecture, Gate policy, runtime maps, and feature registers.
