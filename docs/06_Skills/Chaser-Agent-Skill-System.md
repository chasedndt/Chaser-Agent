# Chaser agent Skill System

Chaser agent skill assets may include skill files, prompt files, workflows, runtime manifests, memory references, and extensions.

```text
Agents may propose skill edits.
Evaluators may score skill edits.
Only validation-gated improvements become candidates.
Only reviewed candidates become active skills.
No uncontrolled self-editing.
```
Skill changes should carry a before/after eval result, provenance, authority risk, and rollback path.
